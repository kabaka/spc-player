# SPC Player

A high-fidelity SNES SPC music player, instrument explorer, and audio workstation — built as a client-side PWA.

## Overview

SPC Player lets you play, analyze, and export music from SNES SPC files with cycle-accurate DSP emulation. Extract individual instruments, toggle tracks, adjust playback speed, and perform with SPC instruments via keyboard or MIDI — all in the browser, fully offline.

## Status

Under active development. Not yet functional.
